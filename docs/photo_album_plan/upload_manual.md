# 写真の追加と管理マニュアル

このドキュメントでは、家族旅行の思い出（写真や動画）を `memories_w7y2z8q.html` に手動で追加・更新するための手順と運用ルールをまとめています。

## 1. 写真の準備と保存
1. 旅行後、ベストショットを数十枚程度ピックアップします。
2. 画像ファイルのサイズが大きい場合は、Webでの表示が重くならないようにファイルサイズを圧縮（長辺1920px程度、1枚あたり数百KB〜1MB程度が目安）しておくことをお勧めします。
3. リポジトリ内の `img/` フォルダ（場合によっては `img/memories/` のように専用フォルダを作ってもOK）に画像を保存します。
   - ファイル名は小文字の英数字にし、スペースは入れないようにしてください。
   - 例: `day1_ocean.jpg`, `day2_water_buffalo.png`

## 2. HTMLファイルの編集
1. `memories_w7y2z8q.html` をエディタ（VS Codeなど）で開きます。
2. 写真を追加したい日程（Day 1 〜 Day 4）のセクションを見つけます。
   - `<div class="photo-grid">` から `</div>` の間に、以下のHTMLのブロックを追加します。

```html
<!-- 追加する写真のテンプレート（ここから） -->
<div class="photo-card" onclick="openModal('img/ファイル名.jpg', 'ここにお好きなコメントを書きます')">
    <img src="img/ファイル名.jpg" alt="画像の説明（代替テキスト）">
    <div class="photo-caption">ここにお好きなコメントを書きます</div>
</div>
<!-- 追加する写真のテンプレート（ここまで） -->
```

### 編集時のポイント
- `openModal` の中身（表示用）と、`<img src="...">` の中身（サムネイル用）の **2箇所** のファイルパスを合わせる必要があります。
- `openModal` の第2引数と、`<div class="photo-caption">` の中身の **2箇所** のコメント内容を合わせる必要があります。
- 順番を変えたい場合は、この `<div class="photo-card">...</div>` のまとまりごと上下に入れ替えてください。

## 3. 確認と公開
1. ローカル環境（自分のPC上）で `memories_w7y2z8q.html` を開き、画像が正しく表示されること、クリックして拡大・コメント表示が機能することを確認します。
2. 問題なければ、変更されたHTMLファイルと、新しく追加した画像ファイル群をGitでコミット・プッシュします。

```bash
git add .
git commit -m "feat: Add photos for memories page"
git push
```

数分後にはGitHub Pagesに反映され、いつもの合言葉ページ経由でご家族全員が見られるようになります！
