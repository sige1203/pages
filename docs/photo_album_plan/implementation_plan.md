# 石垣・竹富・西表 3島周遊 メモリーズ（写真・動画アルバム）実装計画

## 目標
旅行後に、家族で集まって・あるいは各自のスマホから日程ごとの写真や動画を見返し、思い出を語り合えるようなアルバムページを追加する。

## ターゲット・利用シーン
- **利用者:** 家族全員（スマホやタブレットでの閲覧がメインとなる想定）
- **シーン:** 旅行後のリビングでの振り返り、離れた場所にいる祖父母等への共有

## コンセプト
「あの時こうだったね！」と会話が弾みやすいように、単なる写真の羅列ではなく**「旅の時系列（日程・アクティビティごと）」に整理されたストーリー性のあるアルバム**にする。

---

## ページ構成案（メモリーズページ）

### ファイル名
セキュリティを考慮し、こちらもランダム化されたファイル名にする（例: `memories_v9k2p5x.html`）。

### 主要コンポーネント
1. **ヒーローセクション（トップ）**
   - 家族のベストショット1枚を大きく背景に配置。
   - タイトル：「石垣・竹富・西表 2026 Memorial Album」
2. **日程別タイムライン（Day 1 〜 Day 5）**
   - 各日ごとに見出し（例: `Day 2: 竹富島さんぽ & 石垣牛ディナー`）を設ける。
   - スクロールに合わせて、その日の出来事が時系列で表示される。
3. **メディアギャラリー（各日程内）**
   - **写真:** 見やすいグリッド表示（スマホでは2カラム、PCでは3〜4カラム）。タップで拡大（ライトボックス表示）。
   - **動画:** 自動再生・ループは控えめにするか、ユーザーがタップして再生する形式（音が出るため）。
   - **一言コメント（キャプション）:** 写真や動画に対して「水牛の〇〇ちゃん、可愛かったね」「パパ、初めてのダイビングで緊張気味」など、親しみやすい一行コメントを添えられるようにする。
4. **ハイライト（オプション）**
   - 「ベストスマイル賞」「ハプニング大賞」など、家族ならではのネタをピックアップするコーナー。

## 実装ステップ詳細

### フェーズ1：土台となるHTML/CSSの作成（静的モックアップ）
- 新しいHTMLファイル (`memories_xxx.html`) を作成。
- 既存の `index.html` や `3islands_xxx.html` とトンマナ（デザインの雰囲気）を合わせる。Zen Maru Gothicフォント等を踏襲し、温かみのあるデザインにする。
- ダミー画像を用いて、Day1〜Day5までのタイムラインレイアウトと、写真のグリッドレイアウトを組む。
- 写真をタップしたときの拡大表示機能（簡易的なLightbox的なJavaScript）を実装する。

### フェーズ2：セキュリティと導線の追加
- `memories_xxx.html` に、未認証時に `index.html` にリダイレクトするJavaScriptを追加する。
- `index.html` のイベントカード内に、「📷 Memories (思い出アルバム)」ボタンを追加し、この新ページへリンクさせる。
- 検索エンジン避けの `noindex` タグを追加する。

### フェーズ3：実際のメディア（写真/動画）の登録ガイドの整備
- HTML内に直接画像を埋め込む形だと、旅行後に数百枚の写真をアップロードするのが大変になる可能性がある。
- 運用を楽にするため、画像ファイルは `img/memories/day1/` のようなフォルダに配置し、HTML側でそれを読み込むような構成にしておく。
- または、**運用ルール**として、「Googleフォト等の共有アルバムのリンクを、日程ごとに埋め込む」形にするか検討事項とする。

## ユーザー確認が必要な事項
1. **写真の保存先と追加方法**
   - 数枚のベストショットをHTML（リポジトリ）に直接入れてレイアウトする「厳選アルバム形式」にするか？
   - それとも、大量の写真は別途Googleフォトなどに入れ、このページからは「Day1の全写真はこちら」とリンクで飛ばす形式にするか？（厳選数枚だけをこのページに載せるハイブリッドがおすすめです）
2. **デザインの雰囲気**
   - アルバム風（ポラロイド写真のような枠付き）にするか、シンプルでモダンな全画面グリッドにするか？
